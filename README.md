# Interactive-web-interface-
WORKS ONLY IN TABLET MODE 
This project is an interactive web interface that showcases the capabilities of modern front-end development through responsive layout (Flexbox, Media Queries), 
complex CSS animations, and deep DOM manipulation in pure JavaScript. Features include dynamic grid reorganization for different device types, an interactive system 
(cloning elements on click, smart block removal with automatic resizing of adjacent elements), and an HTML5 Canvas graphics module with motion physics and vector rendering of objects via requestAnimationFrame.
